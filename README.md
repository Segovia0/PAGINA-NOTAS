# Sistema de Control de Notas Universitario

Este proyecto consiste en una aplicación web Full-Stack diseñada para gestionar el control de notas, índices académicos y estadísticas estudiantiles en una institución universitaria de manera eficiente, dinámica y modular.

## 🚀 Características y Objetivos del Proyecto
* **Arquitectura Limpia**: Separación rígida de responsabilidades en el Frontend mediante ES6 Modules y en el Backend usando `express.Router()`.
* **Persistencia Robusta**: Guardado sincrónico automatizado en archivo `estudiantes.json`.
* **Reactividad Nativa**: Actualizaciones automáticas del DOM y del panel de estadísticas generales mediante llamadas asíncronas (`fetch`) sin recargar la página.
* **Búsqueda Reactiva**: Filtrado en tiempo real en la interfaz de usuario al tipear la cédula de identidad.
* **Validaciones de Seguridad Doble**: Filtros sanitarios exhaustivos tanto en la capa cliente (JavaScript) como en el servidor.
